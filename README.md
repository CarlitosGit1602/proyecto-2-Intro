DOCUMENTACIÓN DEL PROYECTO
1.	Atributo de Análisis de Problema Problema complejo de ingeniería:
El proyecto se centra en el desarrollo de un sistema interactivo capaz de generar mapas aleatorios conectados,
mover entidades autónomas con reglas diferenciadas y mantener una experiencia de juego estable.
El problema combina algoritmos de generación procedural, búsqueda de caminos, simulación de agentes y control de recursos dentro de un espacio dinámico.
Requiere integrar aspectos matemáticos como manejo de matrices, distancias Manhattan y algoritmos de búsqueda,
así como principios de ingeniería orientados a modularidad, eficiencia y sostenibilidad.
También demanda considerar el impacto computacional del procesamiento continuo, de modo que el sistema sea eficiente y escalable.
